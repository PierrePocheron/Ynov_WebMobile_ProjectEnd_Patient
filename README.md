## 🧱 `Features`
- Setup PWA
- Add connection to Firebase with environment variables
- Deploy app on Netlify
- Setup environnement variables for Netflify
- Form authentication
- Form register
- Setup rooter
- Take appointment (display list of providers & choose date)
- Cancel appointment
- Button to install PWA
- Splash Screen
